## Github 搜索技巧

------

```
//匹配开发语言为Objective-C，star数大于0的
language:Objective-C stars:>0

//匹配用户填写的地址在china的开发者
location:china

//匹配拥有超过一千名关注者的开发者
followers:>=1000

//匹配用户实名为jack的开发者
in:fullname
```
